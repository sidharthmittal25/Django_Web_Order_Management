# Django_Web_Order_Management
Developed a order management system using Django

I have uploaded 4 files:
1. The URL file contains the list of URLs used in views file
2. The index file has the html code and java script for the home page
3. Model has the list of models used in the Django code. This file is for those who want to use Django Object Relation model for developing apps
4. Views has the list of different functions used for developing the app
